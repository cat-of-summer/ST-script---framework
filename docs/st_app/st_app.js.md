# st_app.js

## Описание

Минималистичный реактивный фреймворк на базе Custom Elements (`<st-app>`). Предоставляет реактивность, интерполяцию, двустороннюю привязку форм, директивы условного рендеринга и циклов, вычисляемые свойства (геттеры) и вложенные компоненты.

## Регистрация компонента

```js
App.create(options)
```

| Поле | Описание |
|---|---|
| `app` | Уникальный идентификатор (значение атрибута `::app`) |
| `template` | HTML-шаблон компонента |
| `setup()` | Вызывается после инициализации реактивности |
| `...data` | Произвольные реактивные свойства и методы |
| `get property()` | Вычисляемое свойство (пересчитывается автоматически) |

## Директивы шаблона

| Директива | Описание |
|---|---|
| `{{ expr }}` | Интерполяция JS-выражения |
| `#model="prop"` | Двусторонняя привязка input/select/checkbox |
| `@event="handler()"` | Обработчик события (`$event` — объект события) |
| `#if="cond"` | Условный рендеринг |
| `#else-if="cond"` | Ветка else-if |
| `#else` | Ветка else |
| `#for="item in arr"` | Цикл по массиву (`$index`) или объекту (`$key`) |
| `#show="cond"` | `visibility: hidden` без удаления из DOM |
| `:attr="val"` | Передача данных в дочерний компонент как атрибут |
| `::app="id"` | Указать, какой компонент монтировать |

## Метод экземпляра

```js
this.watch(source, callback, options?)
```

| Параметр | Описание |
|---|---|
| `source` | Строка свойства `'count'`, путь `'user.name'`, вызов `'save()'` или функция-геттер |
| `callback(newVal, oldVal, unwatch)` | Обработчик изменения; `unwatch()` останавливает наблюдение |
| `options.immediate` | Вызвать `callback` сразу при подписке |
| `options.deep` | Глубокое наблюдение за объектом |

## Пример

```html
<st-app ::app="counter"></st-app>

<script>
    App.create({
        app: 'counter',
        count: 0,
        increment() { this.count++; },
        template: `
            <div>
                <p>{{ count }}</p>
                <button @click="increment()">+1</button>
            </div>
        `
    });
</script>
```
