# st_coolie+st_modal.html

## Описание примера

Демонстрация связки классов `st_modal` и `st_cookie`. Промо-модальное окно, которое появляется один раз в день.

### Логика работы

1. Страница загружается.
2. Через `config.delay` (по умолчанию 3 с) `st_cookie.get(config.cookieName)` проверяется: если `null` — модал открывается.
3. Модал можно закрыть кнопкой ×, нажатием Esc или кликом по оверлею.
4. При любом закрытии (`on_hide`) записывается cookie на `config.cookieExpires` секунд (86400 = 1 день).
5. Если модал не закрытD — автозакрывается через `config.autoClose` (300 с = 5 мин), что также триггерит `on_hide`.
6. При перезагрузке страницы cookie есть — модал не появляется.

### Настройка

Весь конфиг собран в один объект `config`:

| Параметр | Значение | Описание |
|---|---|---|
| `cookieName` | `'promo_modal'` | Ключ cookie |
| `cookieExpires` | `86400` | Срок жизни в секундах |
| `delay` | `3` | Задержка появления в секундах |
| `autoClose` | `300` | Автозакрытие в секундах |
| `duration` | `0.35` | Длительность анимации |
| `content` | HTML-строка | Контент модального окна |
| `location` | `'center center'` | Позиция модала |
| `overlay` | `true` | Показывать оверлей |

### Анимация

CSS-класс `.modal-box` масштабируется (появление — scale 0.85 → 1, скрытие — 1 → 0.85 + opacity 0 на `<modal>`).
