# st_mask.js

## Описание

Накладывает маску на `<input>` с поддержкой нескольких альтернативных масок (выбирается лучшая по прогрессу заполнения). Информация о состоянии заполнения хранится в `dataset` поля.

## Конструктор

```js
new st_mask(param)
```

| Параметр | По умолчанию | Описание |
|---|---|---|
| `param.selector` | — | CSS-селектор полей |
| `param.masks` | `[]` | Массив шаблонов маск |
| `param.default_filler` | `'_'` | Символ-заполнитель для пустых динамических позиций |
| `param.placeholder` | `true` | Показывать заполнители до |
| `param.on_input(input, info)` | — | Колбэк при вводе (`info.mask_id`, `info.progress`, `info.is_full`) |

## Публичные методы

| Метод | Возвращает | Описание |
|---|---|---|
| `add_mask(mask)` | `number` | Добавить маску, возвращает индекс |
| `remove_mask(index)` | `void` | Удалить маску по индексу |
| `set_masks(masks)` | `void` | Заменить весь список масок |

## Синтаксис маски

- Статичные символы записываются напрямую: `+7 (`
- Динамические (регексп) символы: `{{регексп}}` — один символ
- Повторение: `{N*{регексп}}` — N символов
- Свой filler: `{{регексп}=X}` — X будет заполнителем вместо `default_filler`

## Данные input.dataset после ввода

| Поле | Описание |
|---|---|
| `dataset.mask_id` | Индекс активной маски |
| `dataset.progress` | Количество заполненных динамических позиций |
| `dataset.is_full` | `"true"` если маска заполнена полностью |

## Пример

```js
const mask = new st_mask({
    selector: 'input[name=PHONE]',
    masks: [
        '+{7} ({3*{\\d}}) {3*{\\d}} - {2*{\\d}} - {2*{\\d}}',
    ],
    on_input: (input, info) => {
        input.style.background = info.is_full ? 'green' : 'red';
    }
});

// Добавить маску позже
const idx = mask.add_mask('{2*{\\d}=D} {2*{\\d}=M} {4*{\\d}=Y}');
mask.remove_mask(idx);
```
